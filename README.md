# Career Chatbot

A simple Python Flask chatbot that recommends career paths, required skills, and personalized learning roadmaps. Users can ask about careers like Data Science, Web Development, AI Engineering, or Digital Marketing, and get step-by-step guidance.

## Features
- Interactive chatbot interface using Flask
- Provides recommended skills for each career
- Shows personalized learning roadmap
- Simple, clean, and deployable UI
- Quick deployment using Gunicorn

## Career Paths Supported
- **Data Science** → Python, Statistics, Machine Learning, SQL  
- **Web Development** → HTML, CSS, JavaScript, Flask/Django  
- **AI Engineer** → Python, Deep Learning, TensorFlow/PyTorch  
- **Digital Marketing** → SEO, Content Marketing, Social Media, Analytics  

## Tech Stack
- Python 3.14
- Flask
- HTML/CSS (Frontend)
- Gunicorn (Production server)
